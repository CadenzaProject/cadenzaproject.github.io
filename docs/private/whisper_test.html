Test Whisper model base.en and MUSDB18

Using Whisper to see if it is a good choice for transcription and then human correction. If it is not good, we would need to do human transcriptions from scratch and probably, boundary corrections.

Challenges:

  • Some lyrics are intrinsically more intelligible than others, which would make it harder for the ASR and humans to transcribe, all the spectrum important for us
  • The segmentation model returns section boundaries that may not be perfect and would need human adjustment.

Process:

  • Selected 4 tracks from MUSDB18 with different intelligibility
  • Pass the mixture through section segmentation using all-in-one
  • Pass each section of the mixture and the ground truth vocals through Whisper model base.en

Next:

  • I want to test the behaviour when using separator model to see how much the performance degrade
  • Would using band filters help? E.g. filter between 100-6000 Hz before whisper/human?
